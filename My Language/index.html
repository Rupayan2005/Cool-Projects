Variable Declaration

eta variableName = value

Declares a variable with a name and assigns it a value.

Examples:
eta x = 10
eta name = "John"
eta sum = x + y

Print Statement

lekh expression

Prints a value, variable, or string to the output.

Examples:
lekh x
lekh "Hello World"
lekh sum

Supported Data Types

  • Numbers: 10, 25, 100
  • Strings: "Hello", "World"
  • Variables: x, y, name
  • Expressions: x + y, a - b

Operators

  • = Assignment
  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
